🚀 Getting Started
bash
# Clone the repository
git clone https://github.com/yourusername/facebook-feed-simulator.git

# Compile the project
cd facebook-feed-simulator
g++ -std=c++17 -o fbsim main.cpp User.cpp Post.cpp FriendSystem.cpp FeedManager.cpp

# Run the simulator
./fbsim  # Linux/macOS
.\fbsim.exe  # Windows
